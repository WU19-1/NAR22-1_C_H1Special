This is a simple fizz buzz problem but with multiple numbers and characters
You are given 2 integers n and k
the second line contains a String s that contains k characters
the third line contains k integers

if the input is an integer, you need to show the last digit only
lets say the number is 112 then you just need to print out 2

e.g.,
Sample Input 
13 3
JLB
2 4 3

Sample Output
1
J
B
JL
5
JB
7
JL
B
J
1
JLB
3
